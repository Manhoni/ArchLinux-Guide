# App uteis para baixar no arch

## Comandos hyprland padrão Arch Linux

### Principais

- Super + Return = abre terminal (kitty)
- Super + E = pastas (thunar)
- SUPER + A - pesquisa apps (wofi)
- Super + Backspace = Fecha janela ativa
- Super + Shift + numero = manda a janela para outra area de trabalho
- Super + numero = muda area de trabalho
- Super + P = aumenta ou diminui largura da janela
- SUPER + F - Fullscreen
- SUPER + X - logoff hyprland


## Apps uteis 

1. leitor pdf - zathura
2. notificações - dunst
3. pastas - thunar
4. bluetooth - bluetoothctl ou cluetoothiu
5. video - mpv
6. imagem - imv
7. banco de dados - mariadb
8. http - apache
9. php - phpmyadmin php
10. navegador - firefox
11. git - git
12. pesquisa app - wofi
13. editor texto - neovim, vim, nano
14. monitor tarefas - btop, mission center
